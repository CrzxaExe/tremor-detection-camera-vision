# Tremor Detection dengan Kamera
## Dataset
Sumber dataset dapat diakses pada [TremorComputerVision](https://github.com/JuliusWelzel/TremorComputerVision).
## Deskripsi Dataset
Amplitudo - Menggambarkan kuat lemahnya getaran tangan.
Frekuensi - Menggambarkan seberapa cepat getaran terjadi per detik.
Stabilitas - Mengukur seberapa konstan gerakan tangan dalam interval tertentu.
